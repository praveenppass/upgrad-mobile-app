# Study Plan Containers

## Container1: Specialization Screen

- Displays the list of study plans (specializations).

## Container2: Study Plan Root

- Shows the course list for an individual study plan.

## Container3: Modules List

- Displays the list of modules within an individual course.

## Container4: Session

- Represents a session within a module.

## Container5: Segment (Topic)

- Represents a segment, also called a topic, within a session.

## Container6: Asset

- Represents an individual asset (content item) within a segment.
